Classroom · stretch goal

One tutor, one class

Every student runs their own MasteryModel — same BKT engine, same dependency graph. This view aggregates those per-student tracers into a single classroom picture, so a teacher can see who's stuck, where, and why, instead of relying on self-reported understanding one-on-one.

This is a simulated class of students working through Mentora's bundled photosynthesis demo lesson, generated locally in your browser to demonstrate how individual mastery models roll up to a class view — no student data is collected or transmitted.
